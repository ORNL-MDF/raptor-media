# Raptor Media

This repository contains images, videos, and other media assets for [Raptor](https://github.com/ORNL-MDF/Raptor).

## Repository Structure

The repository contains:

- `images/` - Contains image files (e.g., PNG, JPEG, SVG) used on the website.
- `videos/` - Contains video files (e.g., MP4, WebM) that are embedded or linked on the website.

## Usage

Reference media in the HTML or Markdown files:

```markdown
![image](https://raw.githubusercontent.com/ORNL-MDF/raptor-media/main/images/Raptor.svg)
```
